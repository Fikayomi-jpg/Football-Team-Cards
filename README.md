# Football Team Cards ⚽

A dynamic and interactive **Football Team Cards web application** built using **HTML, CSS, and JavaScript**.  
This project displays footballer cards and allows users to **filter players by their positions**.

---

## 📌 Features

- Display football player cards with images and details
- Filter players by position (e.g. Forward, Midfielder, Defender, Goalkeeper)
- Dynamic rendering using JavaScript
- Responsive card layout for mobile and desktop
- Clean and modern UI design

---

## 🛠️ Built With

- HTML5
- CSS
- JavaScript (ES6)

---

## 🚀 How to Use

- Clone the repository:
  git clone https://github.com/Fikayomi-jpg/Football-Team-Cards.git

- Open the project folder:
  cd Football-Team-Cards

- Open `index.html` in your browser

---

## 🎯 Features in Detail

- Filter players by clicking position buttons
- Smooth and responsive layout for all screen sizes
